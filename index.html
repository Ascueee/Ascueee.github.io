üìé Race Kart AI | Nov 2023 - Nov 2023

This was a AI system that I developed inspired by Mario Kart with a dynamic way point system for easy to use track creation, that uses linear interpolation and cubic splines to create a path for the kart to follow. üìé Super Mario Galaxy UI and Menus | Sep 2023 - Oct 2023

For this project I decided to pick Super Mario Galaxy and specifically the opening menus of the game. To achieve this a menu manager script was used in combination with LeanTween a animation and lerp tool to recreate the smooth transitions between the different UI elements.

üìé Motion Capture Dark Souls Recreation | Nov 2023 - Dec 2023

Worked with a partner to use motion capture to recreate animations from a game, we decided to pick Darksouls 3 player animations. Worked with setting up, callibrating, recording, and cleaning up motion capture data for in engine use. In addition, implemented the animations onto a model with blend trees and a state change script to smoothly transition animations with a simple button press. This was done also with the addition of inverse kinematics(ik) to polish up the animations on the model and to ground the animations to the terrain.
